# DigitalPalette
![Sample app icon](../src/main/icons/full/icon_full_256.png)

DigitalPalette 是免费开源软件, 您可以使用它创建一组符合一定色彩调和规则的颜色, 并将其应用到您的创作中.

# 版本
1.0.30 (开发版)
